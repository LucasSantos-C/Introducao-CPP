# Introdução a Programação em C/C++

Um repositório com aulas e exercícios simples baseados no livro "Introdução a Programação Orientada e Objetos em C++" - Silva FIlho e
em cursos de programção em C e C++ do canal "Bro Code". Aqui serão contidos pequenos projetos em C e C++ como forma de estudo e prática
de programação para baixo nível.

## Fontes de inspiração:
* Bro Code
* Programming with Mosh
